The extraction prompt

Review our entire conversation. Extract every preference, decision, project detail, architectural choice, and instruction I shared. Ignore questions, debugging, code blocks, and small talk. Write each as a standalone fact useful to a different AI assistant with zero context. Start each with an action word (Use, Prefer, Always, Never, Deploy, etc). Output ONLY this format, nothing else: TRANSFER TO CI: Project: [name of this conversation as shown in your sidebar] - [fact 1] - [fact 2] - [fact 3] ...
